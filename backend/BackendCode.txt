dotenv is a package in Node.js used to load environment variables from a .env file into your application.

 ---{ require("dotenv").config(); }---

This line tells Node.js to:
  > Import the dotenv package
  > Read the .env file
  > Store all variables inside process.env

Why is it Needed?

> In real projects, we should not directly write sensitive data inside code.
Examples:

Database password
API keys
Secret tokens
Email passwords
Port numbers

Instead, we store them in a separate .env file.
